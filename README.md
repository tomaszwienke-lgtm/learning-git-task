# Zadanie: analiza kanałów sprzedaży

Dashboard interaktywny w Pythonie (Dash, Plotly) przedstawiający sprzedaż według kanałów (`Store_type`) w zależności od dnia tygodnia oraz podsumowanie dla każdego kanału.

## Zawartość
- `Zadanie_kanaly_sprzedazy.ipynb` – notebook z pełną implementacją.
- `db/` – folder z danymi źródłowymi (transakcje, klienci, kategorie, kraje).

## Uruchomienie
1. Zainstaluj wymagane biblioteki: `pip install dash plotly pandas`.
2. Uruchom notebook w Jupyter.
3. Po uruchomieniu komórki z aplikacją, otwórz podany link (np. `http://127.0.0.1:8052/`).
