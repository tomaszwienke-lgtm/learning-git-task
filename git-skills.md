# Podsumowanie umiejętności Gita

## Umiejętności które opanowałem:
- git init - tworzenie repozytorium
- git add - dodawanie plików
- git commit - zatwierdzanie zmian
- git push - wysyłanie na GitHub
- git branch - praca na gałęziach
- git merge - łączenie gałęzi
- git clone - klonowanie repozytoriów
